# Vibrant Flight ✈️

Vibrant Flight is a responsive travel website that showcases flight services, destinations, and booking-style UI components. The project focuses on building a clean, modern, and user-friendly interface for exploring travel options.

---

## Features

- Responsive travel website UI
- Flight destinations showcase
- Navigation-based layout
- Booking-style interface design
- Modern frontend styling
- Full-stack project structure

---

## Tech Stack

**Frontend**
- HTML
- CSS
- JavaScript

**Backend**
- Node.js
- Express.js

## Installation and Setup

Clone the repository:

```bash
git clone https://github.com/satishkotipallli/vibrantflight.git
cd vibrantflight
```

Install backend dependencies:

```bash
cd backend
npm install
```

Start the server:

```bash
npm start
```

Open the frontend folder in a browser or run using Live Server.
